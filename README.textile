h1. Important

You will need to import the SDK  from https://github.com/bluebossa63/openstack-java-sdk with components :

openstack-client
keystone-client
keystone-model
swift-client
swift-model

either added as eclipse projects by eclipse build path (with export) or generated by maven project import:


You also need a backend running with the following minimal assets: 

At least two virtual machines :

First vm with:
a reverse proxy 
a JBoss server for identification from https://github.com/bluebossa63/openstack-server-apps
an SMTP server

Second vm with an OpenStack AllInOne with Swift and Keytone services ( using devstack or packstack)
